Que es REACT?

React es una biblioteca de JavaScript de código abierto diseñada para crear interfaces de usuario con el objetivo de facilitar el desarrollo de aplicaciones en una sola página. Es mantenido por Facebook y una comunidad de desarrolladores individuales y empresas. React puede utilizarse como base en el desarrollo de aplicaciones de una sola página o móviles, como Instagram, WhatsApp, Uber, y Facebook.
